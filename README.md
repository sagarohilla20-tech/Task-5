# Laundry Services Website

A modern, responsive single-page website for a professional laundry and dry cleaning service. Built with clean HTML5 and CSS3, demonstrating modern layout techniques like floats, positioning, and responsive design principles.

## 🌟 Features

- **Professional Navbar**: Fixed navigation with logo, menu links, and user greeting
- **Hero Section**: Eye-catching landing area with compelling copy, service highlights, and call-to-action
- **Responsive Design**: Mobile-friendly layout using floats and flexible units
- **Modern Styling**: Clean typography, hover effects, shadows, and professional color scheme
- **Semantic HTML**: Proper use of HTML5 elements for accessibility and SEO

## 🛠️ Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Advanced styling with:
  - Flexbox alternatives (using floats for demonstration)
  - CSS positioning (`relative`, `absolute`)
  - Pseudo-elements (`::before`, `::after`)
  - Modern units (`vh`, `rem`)
  - Transitions and hover effects
  - Z-index layering

## 📁 Project Structure

## 🚀 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No server required - static website

### Running the Project

1. Clone or download the project files
2. Ensure the following files are in the same directory:
   - `index.html`
   - `style.css`
   - `images/logo.webp`
   - `images/image.webp`
3. Open `index.html` in your web browser

The site will load immediately with no build step required.

## 🎨 Design Highlights

- **Color Scheme**: Dark navbar with vibrant blue accents (#0066ff, #0052cc)
- **Typography**: Clean, readable fonts with proper hierarchy
- **Layout Techniques Demonstrated**:
  - Float-based layouts (classic approach)
  - CSS positioning (relative/absolute)
  - Clearfix pattern
  - Inline-block elements
  - Z-index management

## 📱 Responsiveness

The design uses percentage-based widths and viewport units (`vh`) for natural responsiveness. For production use, consider adding media queries for smaller screens.
